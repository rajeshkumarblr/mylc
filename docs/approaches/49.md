### Approach

We can group anagrams by realizing that all anagrams will result in the exact same string if their characters are sorted.
1. Create a Hash Map mapping a `string` (the sorted version of the word) to a `list of strings` (all words that match this sorted version).
2. Iterate through each word in the input array.
3. Sort the characters of the word to create a key.
4. Append the original word to the list in the Hash Map corresponding to this key.
5. Finally, return all the lists from the Hash Map.

**Time Complexity:** **O(N * K log K)** where N is the number of words and K is the max length of a word.
**Space Complexity:** **O(N * K)** to store the map.
