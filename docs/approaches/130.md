To find the surrounded regions of `'O'`s, we can work in reverse: find the unsurrounded regions first.

1. Any `'O'` that is on the border of the board cannot be surrounded. Furthermore, any `'O'` connected to a border `'O'` is also unsurrounded.
2. We can iterate over the border cells. For every `'O'` found on the border, we run a Depth-First Search (DFS) to explore its entire connected island.
3. During the DFS, we temporarily mark all visited `'O'` cells with another character, such as `'T'`, to indicate they are safe from flipping.
4. After completing the DFS for all border cells, we iterate over the entire board one last time:
   - Any remaining `'O'` must be surrounded, so we flip it to `'X'`.
   - Any `'T'` was a safe border-connected cell, so we flip it back to `'O'`.

**Time Complexity:** **O(M * N)** where M and N are the dimensions of the board. We visit each cell a constant number of times.

**Space Complexity:** **O(M * N)** in the worst case for the recursion call stack, if the entire board is filled with `'O'`s.
