To determine if an array contains duplicate elements, we can utilize a Hash Set. 
1. Iterate through each element in the array.
2. Check if the element already exists in the Hash Set.
3. If it does, we found a duplicate, return `true`.
4. Otherwise, insert the element into the Hash Set.
5. If the loop completes without finding any duplicates, return `false`.

**Time Complexity:** **O(N)** where N is the number of elements in the array.

**Space Complexity:** **O(N)** to store the elements in the Hash Set.
