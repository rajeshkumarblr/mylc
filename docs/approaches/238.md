We need to calculate the product of all elements except the current one without using division.
1. We can achieve this by calculating the prefix products and suffix products.
2. First pass: Iterate from left to right. Keep a running product (`prefix`) and store it in our result array.
3. Second pass: Iterate from right to left. Keep a running product (`postfix`) and multiply it with the value already stored in the result array.

**Time Complexity:** **O(N)**

**Space Complexity:** **O(1)** (excluding the output array).
