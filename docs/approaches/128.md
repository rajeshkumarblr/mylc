### Approach

To find the longest consecutive sequence in $\mathcal{O}(N)$ time:
1. Insert all numbers into a Hash Set for $\mathcal{O}(1)$ lookups.
2. Iterate through the numbers in the set.
3. For each number, check if it is the **start of a sequence**. A number is the start of a sequence if `(num - 1)` does NOT exist in the set.
4. If it is the start, keep checking if `(num + 1)`, `(num + 2)`, etc., exist in the set, and count the length of the sequence.
5. Update the maximum length found so far.

**Time Complexity:** $\mathcal{O}(N)$
**Space Complexity:** $\mathcal{O}(N)$ for the Hash Set.
