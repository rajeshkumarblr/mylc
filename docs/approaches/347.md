We can use Bucket Sort to achieve an **O(N)** time complexity.
1. Count the frequency of each element using a Hash Map.
2. Create an array of lists (buckets) where the index represents the frequency of occurrence. The size of this array will be $N + 1$.
3. Iterate through the Hash Map and place each element into the bucket corresponding to its frequency.
4. Iterate through the buckets in reverse order (from highest frequency to lowest) and gather elements until we have K elements.
