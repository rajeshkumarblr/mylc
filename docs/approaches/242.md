### Approach

To check if two strings are anagrams of each other:
1. First, check if their lengths are equal. If not, return `false`.
2. Create an array of size 26 to count the frequencies of each character.
3. Iterate through both strings simultaneously. Increment the count for characters in the first string and decrement for characters in the second string.
4. Finally, iterate through the count array. If any count is not zero, the strings are not anagrams.

**Time Complexity:** $\mathcal{O}(N)$ where $N$ is the length of the strings.
**Space Complexity:** $\mathcal{O}(1)$ since the count array size is fixed at 26.
