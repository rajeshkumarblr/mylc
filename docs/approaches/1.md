### Approach

We can solve this problem in one pass using a Hash Map.
1. Create a Hash Map that maps an element's value to its index in the array.
2. Iterate through the array. For each element, calculate the `complement` required to reach the `target` (i.e., `target - current_element`).
3. Check if this `complement` exists in the Hash Map.
4. If it does, we have found the two numbers. Return their indices.
5. Otherwise, add the current element and its index to the Hash Map.

**Time Complexity:** $\mathcal{O}(N)$
**Space Complexity:** $\mathcal{O}(N)$ for the Hash Map.
