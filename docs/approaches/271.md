To encode and decode a list of strings safely, we need a delimiter that won't naturally occur and cause collisions.
- **Encoding**: For each string, append its length, followed by a special character (e.g., `#`), and then the string itself. Example: `["neet", "code"]` becomes `4#neet4#code`.
- **Decoding**: Read the integer until the `#` to get the length L of the next string. Extract exactly L characters after the `#`. Repeat until the end of the encoded string.


| Component | Complexity | Details |
|---|---|---|
| CPU | **O(N)** | where N is the total length of all strings combined. |
| Memory | **O(1)** | extra space for decoding/encoding. |
