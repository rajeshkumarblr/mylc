# User Customizations & Rules

## Problem Setup Rule
- When the user asks to "setup" a LeetCode problem (e.g., "setup LC 295"), this means setting up the C++ boilerplate code only. Do not write the solution code or start solving it.

## Practice and Error Resolution Rule
- This repository is for the user to practice LeetCode solutions. When the user asks for help with an error (e.g., infrastructure issue, compilation error, runtime error), ONLY fix the specific error or issue at hand.
- DO NOT provide the complete solution or fix the underlying logical algorithm unless the user EXPLICITLY asks for the solution. Let the user struggle with the algorithm logic themselves to learn.
